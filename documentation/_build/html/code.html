class SmartGrazer.SmartGrazer(params)¶

Bases: object

This class represents loads, configures and executes the softwares’ compontents.

generator = None¶

The generator instance dynamically assigned depending on the configuration.

run()¶

This function executes the cli command.

The default action will generate payloads depending on the stored configuration.

When configuration is changed, the handleConfiguration method will be triggered.

SmartGrazer.main()¶

This Function parses the command-line arguments and initiates the SmartGrazer software.

General usage:

Command:SmartGrazer [options]

Full list of options available through:

Command:SmartGrazer -h
class config.Configuration.Configuration¶

Bases: object

This class manages the configuration-files of the application.

This includes the main configuration for the software and the configuration of the built-in payload generator.

All changes will be stored in a file called config.ini in the applications root directory.

class generator.dharma.grammar.Generator.Generator¶

Bases: object

This class represents the context free grammar generator from the open source project dharma.

This software is using this generator in combination with the xss grammar file “xss.dg”.

For more information visit: https://github.com/MozillaSecurity/dharma
